# cv

> Source for my CV

My personal cv built with LaTex. [See the result](https://jneidel.com/cv.en).

## Build

**To build once:**

```sh
make all

# on their own
make de
make en
```

**To build continuously:**

```sh
make watch

# on their own
make de_watch
make en_watch
```

## Dependencies

```sh
yay -S texlive-core
```
